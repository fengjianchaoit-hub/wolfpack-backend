# 狼牙团队工作规范

**生效日期**: 2026-03-03  
**制定人**: 老板  
**执行监督**: 狼头

---

## 一、核心工作原则

### 1.1 执行 → 检查 → 汇报

**严格执行以下流程，缺一不可：**

```
接收任务 → 执行工作 → 自检确认 → 狼头审核 → 向老板汇报
     ↑                                    ↓
     └──────── 有问题打回重做 ←──────────┘
```

### 1.2 具体要求

| 步骤 | 责任人 | 动作 | 标准 |
|------|--------|------|------|
| 1. 接收任务 | 狼牙成员 | 确认需求 | 不清楚立刻问，不许猜 |
| 2. 执行工作 | 狼牙成员 | 按要求完成 | 多反思，多给观点，不只执行 |
| 3. **自检确认** | 狼牙成员 | 检查结果质量 | 数据对不对、格式对不对、有没有错别字 |
| 4. **狼头审核** | 狼头 | 全面检查 | 每一处都检查，不合格打回 |
| 5. **向老板汇报** | 狼头 | 确认无误后汇报 | 附带检查结果说明 |

**⚠️ 严禁跳过检查环节直接汇报！**

## 二、各岗位工作标准

### 2.1 狼牙01 - 文案编辑

**职责**：
- 数据整理、文案编辑
- 日报生成、报告优化

**交付标准**：
- ✅ 文案不堆砌，有结论、有洞察
- ✅ 无错别字，格式统一
- ✅ 数据核对无误
- ✅ 自检清单逐项确认

**自检清单**（每项必须打勾）：
- [ ] 数据来源是否准确
- [ ] 关键数字是否核对
- [ ] 文案是否有核心观点
- [ ] 格式是否符合要求
- [ ] 有无错别字/语病

---

### 2.2 狼牙02 - 数据分析

**职责**：
- 数据分析、统计建模
- 竞品对比、预测分析

**交付标准**：
- ✅ 分析不罗列，有模型、有预测
- ✅ 数据来源可追溯
- ✅ 结论有数据支撑
- ✅ 自检清单逐项确认

**自检清单**（每项必须打勾）：
- [ ] 数据清洗是否完成
- [ ] 分析逻辑是否通顺
- [ ] 结论是否有洞察
- [ ] 图表数据是否对应
- [ ] 预测是否有依据

---

### 2.3 狼牙03 - 可视化

**职责**：
- 图表生成、可视化设计
- 仪表盘搭建、交互组件

**交付标准**：
- ✅ 图表不花哨，有逻辑、有重点
- ✅ 视觉层次清晰
- [ ] 数据展示准确
- ✅ 自检清单逐项确认

**自检清单**（每项必须打勾）：
- [ ] 图表数据是否准确
- [ ] 视觉重点是否突出
- [ ] 配色是否符合规范
- [ ] 交互是否正常
- [ ] 不同设备显示是否正常

---

### 2.4 狼头 - 团队负责人

**职责**：
- 统筹管理、任务分配
- 审核把关、向老板汇报

**审核标准**：
- ✅ 不走过场，每一处都检查
- ✅ 对最终交付质量负全责
- ✅ 不合格坚决打回重做

**审核清单**：
- [ ] 狼牙自检是否完成
- [ ] 交付物是否符合标准
- [ ] 数据/事实是否准确
- [ ] 格式是否规范
- [ ] 是否可以直接向老板展示

---

## 三、错误处理流程

### 3.1 发现问题

1. **自检发现问题** → 自己修正 → 重新自检
2. **狼头审核发现问题** → 打回重做 → 狼牙修正 → 重新审核
3. **老板发现问题** → 狼头负责 → 团队复盘 → 更新规范

### 3.2 复盘记录

每次错误必须记录：
- 错误描述
- 原因分析
- 改进措施
- 责任人
