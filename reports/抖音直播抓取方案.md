# 抖音直播互动数据抓取方案

## 方案A: 自动化脚本（需要本地运行）

### 1. 安装依赖
```bash
pip3 install websockets requests
```

### 2. 设置 Cookie 环境变量
```bash
export DOUYIN_COOKIE="sessionid=你的sessionid; sessionid_ss=你的sessionid_ss; sid_tt=你的sid_tt; uid_tt=你的uid_tt"
```

### 3. 运行抓取
```bash
python3 douyin_live_grabber.py --room 444502128478
```
